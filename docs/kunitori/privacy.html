Privacy Policy

プライバシーポリシー

国織りにおける情報の取り扱いについて説明します。

端末内に保存する情報

本アプリは、プレイヤーの表示名、ローカル対戦の進行状況、オンライン対戦の 再接続に必要な部屋情報および匿名認証情報を利用者の端末内に保存します。 ローカル対戦の進行状況がオンライン対戦用サーバーへ送信されることはありません。

外部サービスへ送信する情報

本アプリは、オンライン機能と中断した部屋への復帰確認を提供するため、 Firebase Authenticationによる匿名認証を利用します。オンライン対戦を初めて 利用したときにランダムな匿名ユーザーIDが発行され、以後は端末から復元されます。 オンライン対戦を利用しない限り、新しい匿名ユーザーIDは発行しません。氏名やメールアドレスの 登録は必要ありません。認証処理では、FirebaseがIPアドレスやユーザーエージェントを セキュリティおよび不正利用防止のために処理する場合があります。

オンライン対戦を利用した場合は、プレイヤーの表示名、匿名ユーザーID、 部屋コード、席順、参加者と接続状態、盤面、手札、サイコロの出目、操作内容、 スタンプなど、対局の進行に必要なゲームデータをFirebase Realtime Databaseへ 送信します。

本アプリは、氏名、住所、電話番号、メールアドレス、位置情報、連絡先、写真、 広告IDの入力または提供を求めません。広告、アクセス解析、クラッシュ収集のSDKは 使用していません。

利用目的

  • オンライン対戦の部屋作成、参加、対局状態の同期
  • 切断後の再接続および席の所有者確認
  • 不正操作の防止とサービスの安全性確保
  • お問い合わせ対応と不具合調査

第三者サービス

認証およびオンライン対戦データの保存・通信には、Google LLCが提供する Firebase AuthenticationとFirebase Realtime Databaseを利用します。 Firebaseにおけるデータ処理の詳細は、 Firebaseのプライバシーとセキュリティ をご確認ください。

オンライン対戦データの共有範囲

対局に必要な情報は、同じ部屋の参加者へ同期されます。表示名、席構成、接続状態などの ロビー情報は、部屋コードを知る認証済み利用者から閲覧される場合があります。 部屋コードは信頼できる参加者とのみ共有してください。

保存期間と削除

接続状態は切断時に削除され、部屋情報と対局状態は通常、ホストが部屋を退出したときに 削除されます。通信障害やアプリの強制終了などで削除処理が完了しなかった場合は、 削除処理が再度実行されるまで残ることがあります。

タイトル画面の「プライバシーとデータ」から「匿名オンラインIDを削除」を選ぶと、 この端末に紐づくFirebase上の匿名認証記録を削除できます。保存済みのホスト部屋が ある場合は、その部屋情報と対局状態を先に削除します。削除後はオンライン対局への復帰が できず、元に戻せません。次にオンライン対戦を利用すると、新しい匿名ユーザーIDが発行されます。

ゲストとして参加した共有部屋のデータは、対局を継続する他の参加者にも必要なため、 ゲストの匿名ユーザーIDを削除した時点では部屋から個別に削除されない場合があります。 その部屋全体はホストが退出または匿名ユーザーIDを削除したときに削除されます。

アプリの削除またはアプリデータの消去だけでは、端末から匿名認証情報が失われ、Firebase上の 認証記録をアプリから特定できなくなる場合があります。先にアプリ内の削除機能を利用してください。 端末内の表示名とローカル対戦データは、アプリ内で対局を破棄するか、アプリの削除または アプリデータの消去により削除できます。

削除とお問い合わせ

匿名認証記録の削除は、タイトル画面の「プライバシーとデータ」から利用者自身で実行できます。 削除機能の不具合や本ポリシーに関する一般的なお問い合わせは、 GitHub Issues からご連絡ください。GitHub Issuesは公開されるため、匿名オンラインID、部屋コード、 表示名その他の個人を識別し得る情報は投稿しないでください。公開情報だけでは削除対象を 安全に特定できないため、GitHub Issuesで個別データの削除依頼は受け付けません。

本ポリシーの変更

取り扱う情報や利用目的に変更がある場合は、本ページを更新します。 重要な変更は、アプリ内またはストアの更新情報でお知らせします。
