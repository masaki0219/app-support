Privacy Policy

プライバシーポリシー

国織りにおける情報の取り扱いについて説明します。

端末内に保存する情報

本アプリは、プレイヤーの表示名、ローカル対戦の進行状況、オンライン対戦の 再接続に必要な部屋情報および匿名認証情報を利用者の端末内に保存します。 ローカル対戦の進行状況がオンライン対戦用サーバーへ送信されることはありません。

外部サービスへ送信する情報

本アプリは、オンライン機能と中断した部屋への復帰確認を提供するため、 Firebase Authenticationによる匿名認証を利用します。アプリの起動時に、 ランダムな匿名ユーザーIDが発行または復元されます。氏名やメールアドレスの 登録は必要ありません。認証処理では、FirebaseがIPアドレスやユーザーエージェントを セキュリティおよび不正利用防止のために処理する場合があります。

オンライン対戦を利用した場合は、プレイヤーの表示名、匿名ユーザーID、 部屋コード、席順、参加者と接続状態、盤面、手札、サイコロの出目、操作内容、 スタンプなど、対局の進行に必要なゲームデータをFirebase Realtime Databaseへ 送信します。

本アプリは、氏名、住所、電話番号、メールアドレス、位置情報、連絡先、写真、 広告IDの入力または提供を求めません。広告、アクセス解析、クラッシュ収集のSDKは 使用していません。

利用目的

  • オンライン対戦の部屋作成、参加、対局状態の同期
  • 切断後の再接続および席の所有者確認
  • 不正操作の防止とサービスの安全性確保
  • お問い合わせ対応と不具合調査

第三者サービス

認証およびオンライン対戦データの保存・通信には、Google LLCが提供する Firebase AuthenticationとFirebase Realtime Databaseを利用します。 Firebaseにおけるデータ処理の詳細は、 Firebaseのプライバシーとセキュリティ をご確認ください。

オンライン対戦データの共有範囲

対局に必要な情報は、同じ部屋の参加者へ同期されます。表示名、席構成、接続状態などの ロビー情報は、部屋コードを知る認証済み利用者から閲覧される場合があります。 部屋コードは信頼できる参加者とのみ共有してください。

保存期間と削除

接続状態は切断時に削除され、部屋情報と対局状態は通常、ホストが部屋を退出したときに 削除されます。通信障害やアプリの強制終了などで削除処理が完了しなかった場合は、 運営者による削除または削除依頼への対応まで残ることがあります。

匿名認証情報は端末内に保存され、アプリの削除またはアプリデータの消去により端末から 失われます。Firebase上の匿名認証記録は、運営者が削除するまで保持される場合があります。 端末内の表示名とローカル対戦データは、アプリ内で対局を破棄するか、アプリの削除または アプリデータの消去により削除できます。

削除依頼とお問い合わせ

オンライン対戦データまたは匿名認証記録の削除を希望する場合や、本ポリシーに関する お問い合わせは、 GitHub Issues からご連絡ください。対局データの削除依頼では、対象を確認できる範囲で部屋コード、 表示名、利用したおおよその日時をお知らせください。本人確認や対象データの特定が できない場合は、削除できないことがあります。

本ポリシーの変更

取り扱う情報や利用目的に変更がある場合は、本ページを更新します。 重要な変更は、アプリ内またはストアの更新情報でお知らせします。
