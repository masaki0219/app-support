GitHub Issuesを開く

よくある質問

1台で遊ぶことはできますか？

はい。「1台で遊ぶ（ローカル対戦）」から2〜4人の対局を始められます。 AIの大名を加えて遊ぶこともできます。

オンライン対戦はどうやって始めますか？

ホストが「オンラインで部屋を作る」を選び、表示された6文字の部屋コードを 参加者へ共有してください。参加者は「部屋に参加する」から同じコードを入力します。 オンライン対戦にはインターネット接続が必要です。

オンライン対戦中に通信が切れた場合はどうなりますか？

アプリは自動的に再接続を試みます。ホストには中断した部屋への復帰案内が 表示される場合があります。復帰できない場合は、新しい部屋を作成してください。

ローカル対戦の続きから遊べますか？

対局中の状態は端末内へ自動保存されます。ホーム画面に「続きから」が表示された場合は、 中断した対局を再開できます。対局を破棄した場合やアプリを削除した場合は復元できません。

オンライン対戦のデータを削除できますか？

通常、部屋のデータはホストが部屋を退出したときに削除されます。 残存データの削除を希望する場合は、プライバシーポリシーをご確認のうえ、 GitHub Issuesからご連絡ください。
