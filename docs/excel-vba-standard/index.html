よくある質問

Excel VBA Basic 問題集との違いは何ですか？

本アプリは、変数と配列、プロシージャ、ステートメント、ワークシート関数の利用、 検索とオートフィルター、並べ替え、テーブル操作、ファイル操作、エラー対策、 デバッグといった応用分野を扱います。基礎の用語や記録マクロ中心の内容は 「Excel VBA Basic 問題集」でお試しください。

学習履歴はどこに保存されますか？

問題ごとの正解数、不正解数、最後に解いた日時などの学習履歴は、 原則として利用者の端末内に保存されます。

プレミアム機能を復元できますか？

App Storeで購入したプレミアム機能は、同一のApple IDで復元できます。 復元できない場合は、購入時のApple IDでサインインしているか確認してください。

特定の試験の問題が含まれていますか？

本アプリはExcel VBAの応用・実務レベルの学習を目的としたクイズアプリです。収録問題は 学習用に作成したオリジナル問題であり、特定の試験の出題内容や合格を 保証するものではありません。
